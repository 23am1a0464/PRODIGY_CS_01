Caesar Cipher Implementation
Overview
This project implements the Caesar Cipher algorithm in Python. The Caesar Cipher is one of the
simplest and most widely known encryption techniques. It works by shifting each letter in the plaintext
by a fixed number of positions in the alphabet.
Features
Encrypts text using a user-defined shift value.
Decrypts encrypted text back to the original message.
Supports both uppercase and lowercase letters.
Preserves spaces, numbers, and special characters.
How It Works
The user enters a message.
The user enters a shift value.
Each alphabet character is shifted by the specified value.
The encrypted message is displayed.
The program can also decrypt the message using the same shift value.
Example
Input: Message: HELLO Shift: 3
Encrypted Output: KHOOR
Decrypted Output: HELLO
